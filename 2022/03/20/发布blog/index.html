The Birth of a blog

发表于2022-03-20更新于2022-03-21

记录自己创建blog

一 :下载

nodejs

下载链接nodejs

git

下载链接git

hexo

执行命令npm install express -g

测试命令

检测安装成功与否

node -v

npm -v

git –version

hexo -v

二 建立仓库和生成ssh

  1. 在自己的GitHub里创建一个repository，命名为yourname.github.io

  2. 生成ssh:ssh-keygen -t rsa -C “邮箱地址”

    .ssh生成路径：C:\Users\yourname.ssh

    测定ssh是否绑定成功：ssh -T git@github.com

  3. copy .ssh文件夹内id_rsa.pub的内容

  4. 复制到GitHub>setting>SSH>new SSH key

三 :随便一个文件夹

1. hexo init

初始化并会生成一些文件

.gitbhub
node_modeles
scaffolds
source
themes
.gitignore (or npmignore)
_config.landscape.yml
_config.yml
package.json
package-lock.json 
完成标志  INFO  Install dependencies  ；INFO  Start blogging with Hexo!

​

2. hexo s

本地发布

3. 编辑_config.yml

deploy
type: git
repository: yoursite
branch: main

4. npm install hexo-deployer-git –save

安装hexo-deployer-git自动部署发布工具

5.hexo g

生成网站静态文件到默认设置的 public 文件夹

6. hexo d

发布，如果没有3步，提示Validating config ; 生成.deploy_git

浏览器可以访问

错误提示

5时fatal: unable to auto-detect email address 参考link
